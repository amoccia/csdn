CVE-2026-6599 | langflow-ai langflow fino a 1.8.3 Model Context Protocol Configuration API mcp_projects.py get_client_ip/install_mcp_config X-Forwarded-For escalationi di privilegi

È stata scoperta una vulnerabilità classificata come critico in langflow-ai langflow fino a 1.8.3. Si considera coinvolta una funzione sconosciuta nel file src/backend/base/langflow/api/v1/mcp_projects.py nel componente Model Context Protocol Configuration API. Mediante la manipolazione del parametro X-Forwarded-For di un input sconosciuto conseguenza di una vulerabilità di classe escalationi di privilegi. CVE-2026-6599 è identificato come punto debole. L'attacco può avvenire nella rete. Inoltre, è presente un exploit disponibile. Il miglior modo suggerito per attenuare il problema è applicare una più restrittiva configurazione del firewall.

Source: https://vuldb.com/it/vuln/358234
