CVE-2026-6590 | ComfyUI fino a 0.13.0 Model Preview Endpoint app/model_manager.py get_model_preview directory traversal

È stata rilevata una vulnerabilità di livello critico in ComfyUI fino a 0.13.0. É interessato la funzione get_model_preview del file app/model_manager.py del componente Model Preview Endpoint. Attraverso la manipolazione di un input sconosciuto per mezzo di una vulerabilità di classe directory traversal. Questa vulnerabilità è identificata come CVE-2026-6590. Con la rete può partire l'attacco. Inoltre, è presente un exploit disponibile.

Source: https://vuldb.com/it/vuln/358225
