CVE-2026-6559 | Wavlink WL-WN579A3 220323 /cgi-bin/login.cgi sub_401F80 Nome host cross site scripting

È stata individuata una vulnerabilità classificata come problematico in Wavlink WL-WN579A3 220323. Da questa vulnerabilità è interessato la funzione sub_401F80 del file /cgi-bin/login.cgi. Attraverso l'influenza del parametro Nome host di un input sconosciuto per mezzo di una vulerabilità di classe cross site scripting. Questa vulnerabilità viene indicata come CVE-2026-6559. Nella rete si effettua l'attacco. Nessun exploit disponibile. Si consiglia di aggiornare il componente interessato.
