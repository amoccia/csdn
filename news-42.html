CVE-2025-52641 | HCL AION 2.0 File rivelazione di informazioni (KB0130007)

È stata rilevata una vulnerabilità classificata come problematico in HCL AION 2.0. Risulta coinvolta una funzione sconosciuta del componente File Handler. Attraverso la manipolazione di un input sconosciuto per mezzo di una vulerabilità di classe rivelazione di informazioni. CVE-2025-52641 è identificato come punto debole. L'attacco deve essere fatto localmente. Nessun exploit disponibile.
