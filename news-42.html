CVE-2026-6490 | QueryMine sms fino a 7ab5a9ea196209611134525ffc18de25c57d9593 GET Request Parameter admin/deletecourse.php ID iniezione SQL

In QueryMine sms fino a 7ab5a9ea196209611134525ffc18de25c57d9593 stata rilevata una vulnerabilità di livello critico. Da questa vulnerabilità è interessato una funzione sconosciuta del file admin/deletecourse.php del componente GET Request Parameter Handler. Mediante la manipolazione del parametro ID di un input sconosciuto conseguenza di una vulerabilità di classe iniezione SQL. Questa vulnerabilità è segnalata come CVE-2026-6490. L'attacco si effettua con la rete. Inoltre, è presente un exploit disponibile. Il prodotto segue una rolling release per offrire una consegna continua. Perciò, non sono reperibili dettagli sulle versioni impattate o aggiornate.
