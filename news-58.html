CVE-2026-3842 | QEMU KVM hw/hyperv/syndbg.c cpu_physical_memory_map buffer overflow

È stata scoperta una vulnerabilità classificata come critico in QEMU. Si considera coinvolta una funzione sconosciuta nel file hw/hyperv/syndbg.c nel componente KVM. Mediante la manipolazione di un input sconosciuto conseguenza di una vulerabilità di classe buffer overflow. Questa vulnerabilità è conosciuta come CVE-2026-3842. Nessun exploit disponibile. È consigliato applicare una patch per mitigare questo problema.

Source: https://vuldb.com/it/vuln/358018
