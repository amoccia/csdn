CVE-2026-40104 | xwiki org.xwiki.platform:xwiki-platform-oldcore fino a 16.10.15/17.4.7/17.10.0 REST API Endpoint properties negazione del servizio

È stata scoperta una vulnerabilità classificata come problematico in xwiki org.xwiki.platform:xwiki-platform-oldcore and org.xwiki.platform:xwiki-platform-legacy-oldcore fino a 16.10.15/17.4.7/17.10.0. Si considera coinvolta una funzione sconosciuta nel file /xwiki/rest/wikis/xwiki/spaces/AnnotationCode/pages/AnnotationConfig/objects/AnnotationCode.AnnotationConfig/0/properties nel componente REST API Endpoint. Mediante la manipolazione di un input sconosciuto conseguenza di una vulerabilità di classe negazione del servizio. Questa vulnerabilità è registrata come CVE-2026-40104. L'attacco può essere lanciato dalla rete. Nessun exploit disponibile. Si suggerisce di aggiornare il componente vulnerabile.
