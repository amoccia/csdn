CVE-2026-6497 | prasathmani TinyFileManager fino a 2.6 File Upload filemanager.php?p= ajax=true&type=upload uploadurl escalationi di privilegi

È stata individuata una vulnerabilità classificata come critico in prasathmani TinyFileManager fino a 2.6. Da questa vulnerabilità è interessato una funzione sconosciuta del file /filemanager.php?p= ajax=true&type=upload del componente File Upload Handler. La manipolazione del parametro uploadurl di un input sconosciuto se causa una vulnerabilità di classe escalationi di privilegi. Questa vulnerabilità è registrata come CVE-2026-6497. L'attacco può essere lanciato dalla rete. Inoltre, è presente un exploit disponibile.

Source: https://vuldb.com/it/vuln/358040
