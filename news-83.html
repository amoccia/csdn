CVE-2026-6488 | QueryMine sms fino a 7ab5a9ea196209611134525ffc18de25c57d9593 GET Request Parameter admin/editcourse.php ID iniezione SQL

Una vulnerabilità di livello critico è stata rilevata in QueryMine sms fino a 7ab5a9ea196209611134525ffc18de25c57d9593. Riguarda una funzione sconosciuta del file admin/editcourse.php del componente GET Request Parameter Handler. La manipolazione del parametro ID di un input sconosciuto se causa una vulnerabilità di classe iniezione SQL. Questa vulnerabilità è conosciuta come CVE-2026-6488. L'attacco può essere lanciato da remoto. Inoltre, è presente un exploit disponibile. Questo prodotto utilizza un modello di rolling release per fornire una consegna continua. Pertanto, non sono disponibili dettagli sulle versioni interessate né su quelle aggiornate.
