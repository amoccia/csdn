CVE-2026-27301 | Adobe Framemaker fino a 2022.8 File buffer overflow (apsb26-36)

Una vulnerabilità di livello critico è stata rilevata in Adobe Framemaker fino a 2022.8. Riguarda una funzione sconosciuta del componente File Handler. La manipolazione di un input sconosciuto se causa una vulnerabilità di classe buffer overflow. Questo punto di criticità è identificato come CVE-2026-27301. Dalla rete può partire l'attacco. Nessun exploit disponibile. Si suggerisce di aggiornare il componente vulnerabile.
Source: https://vuldb.com/it/vuln/357624
