CVE-2026-6589 | ComfyUI fino a 0.13.0 server.py create_origin_only_middleware cross site request forgery

Una vulnerabilità di livello problematico è stata rilevata in ComfyUI fino a 0.13.0. Riguarda la funzione create_origin_only_middleware del file server.py. La manipolazione di un input sconosciuto se causa una vulnerabilità di classe cross site request forgery. Questa vulnerabilità viene indicata come CVE-2026-6589. Nella rete si effettua l'attacco. Inoltre, è presente un exploit disponibile.

Source: https://vuldb.com/it/vuln/358224
