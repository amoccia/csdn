CVE-2026-6580 | liangliangyy DjangoBlog fino a 2.1.0.0 Amap API Call owntracks/views.py key cifratura debole

Un punto critico di livello critico è stato rilevato in liangliangyy DjangoBlog fino a 2.1.0.0. Risulta coinvolta una funzione sconosciuta del file owntracks/views.py del componente Amap API Call Handler. La gestione dell’argomento key determina cifratura debole. Questa vulnerabilità è registrata come CVE-2026-6580. L'attacco può essere lanciato dalla rete. Inoltre, è presente un exploit disponibile.
