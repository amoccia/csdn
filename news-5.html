CVE-2026-5785 | Zoho ManageEngine PAM360 Query Report iniezione SQL

Una vulnerabilità di livello critico è stata rilevata in Zoho ManageEngine PAM360 and ManageEngine Password Manager Pro. Riguarda una funzione sconosciuta del componente Query Report Module. Attraverso l'influenza di un input sconosciuto per mezzo di una vulerabilità di classe iniezione SQL. Questa vulnerabilità è identificata come CVE-2026-5785. Con la rete può partire l'attacco. Nessun exploit disponibile. Si suggerisce di aggiornare il componente vulnerabile.

Source: https://vuldb.com/it/vuln/357924
