CVE-2026-22616 | Eaton IPP Software fino a 1.x Web Interface rivelazione di informazioni (EUVD-2026-23175)

Un punto di criticita di livello problematico è stato rilevato in Eaton IPP Software fino a 1.x. Riguarda una funzione sconosciuta del componente Web Interface. Mediante la manipolazione di un input sconosciuto conseguenza di una vulerabilità di classe rivelazione di informazioni. Questa vulnerabilità è registrata come CVE-2026-22616. L'attacco può essere lanciato dalla rete. Nessun exploit disponibile. Si raccomanda di procedere con l’aggiornamento del componente interessato.
