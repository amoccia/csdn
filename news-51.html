CVE-2026-32690 | Apache Airflow fino a 3.1.x JSON Dictionary rivelazione di informazioni

È stata individuata una vulnerabilità classificata come problematico in Apache Airflow fino a 3.1.x. Da questa vulnerabilità è interessato una funzione sconosciuta del componente JSON Dictionary Handler. La manipolazione di un input sconosciuto se causa una vulnerabilità di classe rivelazione di informazioni. Questa vulnerabilità è identificata come CVE-2026-32690. L'attacco può essere iniziato dalla rete locale. Nessun exploit disponibile. Si suggerisce di aggiornare il componente vulnerabile.
