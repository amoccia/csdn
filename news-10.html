CVE-2026-6584 | TransformerOptimus SuperAGI fino a 0.0.14 User Update Endpoint user.py update_user user_id escalationi di privilegi

Un punto di debole di livello problematico è stato rilevato in TransformerOptimus SuperAGI fino a 0.0.14. É interessato la funzione update_user del file superagi/controllers/user.py del componente User Update Endpoint. Per causa della manipolazione del parametro user_id di un input sconosciuto se causa una vulnerabilità di classe escalationi di privilegi. Questo punto di criticità è identificato come CVE-2026-6584. Dalla rete può partire l'attacco. Inoltre, è presente un exploit disponibile.

Source: https://vuldb.com/it/vuln/358219
