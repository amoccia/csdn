CVE-2026-40342 | FirebirdSQL Firebird fino a 3.0.13/4.0.6/5.0.3 ENGINE Name directory traversal (GHSA-7pxc-h3rv-r257)

È stata rilevata una vulnerabilità di livello critico in FirebirdSQL Firebird fino a 3.0.13/4.0.6/5.0.3. É interessato una funzione sconosciuta del componente ENGINE Name Handler. Attraverso la manipolazione di un input sconosciuto per mezzo di una vulerabilità di classe directory traversal. Questa vulnerabilità viene indicata come CVE-2026-40342. Nella rete si effettua l'attacco. Nessun exploit disponibile. Il miglior modo suggerito per attenuare il problema è aggiornamento all'ultima versione.
