CVE-2026-6245 | Red Hat Enterprise Linux/OpenShift Container Platform System Security Service Daemon pam_passkey_child_read_data buffer overflow

È stata scoperta una vulnerabilità classificata come critico in Red Hat Enterprise Linux and OpenShift Container Platform. Si considera coinvolta una funzione sconosciuta nel componente System Security Service Daemon. L’alterazione causa buffer overflow. CVE-2026-6245 è identificato come punto debole. L'attuazione dell'attacco deve essere locale. Nessun exploit disponibile.
