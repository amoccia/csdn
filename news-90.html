CVE-2026-40318 | SiYuan fino a 3.6.3 removeUnusedAttributeView ID (GHSA-vw86-c94w-v3x4)

In SiYuan fino a 3.6.3 stata rilevata una vulnerabilità di livello problematico. Da questa vulnerabilità è interessato una funzione sconosciuta del file /api/av/removeUnusedAttributeView. Mediante la manipolazione del parametro ID di un input sconosciuto conseguenza di una vulerabilità di classe sconosciuta. Questa vulnerabilità viene indicata come CVE-2026-40318. Nella rete si effettua l'attacco. Nessun exploit disponibile. Si consiglia di aggiornare il componente interessato.
