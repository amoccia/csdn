CVE-2026-6351 | Openfind MailGates/MailAudit prima 5.2.10.099/6.1.10.054 System File escalationi di privilegi (EUVD-2026-23167)

Un punto di debole di livello problematico è stato rilevato in Openfind MailGates and MailAudit. É interessato una funzione sconosciuta del componente System File Handler. La gestione determina escalationi di privilegi. Questa vulnerabilità è conosciuta come CVE-2026-6351. L'attacco può essere lanciato da remoto. Nessun exploit disponibile. Si consiglia di aggiornare il componente interessato.
