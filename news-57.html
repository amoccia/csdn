CVE-2026-3599 | imprintnext Riaxe Product Customizer Plugin fino a 2.1.2 su WordPress REST API Endpoint add-item-to-cart product_data options iniezione SQL

In imprintnext Riaxe Product Customizer Plugin fino a 2.1.2 stata rilevata una vulnerabilità di livello critico. Da questa vulnerabilità è interessato la funzione product_data del file /wp-json/InkXEProductDesignerLite/add-item-to-cart del componente REST API Endpoint. Mediante la manipolazione del parametro options di un input sconosciuto conseguenza di una vulerabilità di classe iniezione SQL. Questo punto di criticità è identificato come CVE-2026-3599. Dalla rete può partire l'attacco. Nessun exploit disponibile. Il miglior modo suggerito per attenuare il problema è aggiornamento all'ultima versione.
