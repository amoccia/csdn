CVE-2026-40339 | gphoto libgphoto2 fino a 2.5.33 camlibs/ptp2/ptp-pack.c ptp_unpack_Sony_DPD rivelazione di informazioni (GHSA-42cm-m9hc-r7q8 / EUVD-2026-23607)

È stata rilevata una vulnerabilità di livello critico in gphoto libgphoto2 fino a 2.5.33. É interessato la funzione ptp_unpack_Sony_DPD del file camlibs/ptp2/ptp-pack.c. Attraverso la manipolazione di un input sconosciuto per mezzo di una vulerabilità di classe rivelazione di informazioni. Questa vulnerabilità è segnalata come CVE-2026-40339. Nessun exploit disponibile. È consigliato applicare una patch per mitigare questo problema.
