CVE-2026-6385 | FFmpeg MPEG-PS VOB Media File buffer overflow

È stata scoperta una vulnerabilità classificata come problematico in FFmpeg. Si considera coinvolta una funzione sconosciuta nel componente MPEG-PS VOB Media File Handler. L’alterazione causa buffer overflow. Questo punto di criticità è identificato come CVE-2026-6385. Dalla rete può partire l'attacco. Nessun exploit disponibile.
Source: https://vuldb.com/it/vuln/357862
