CVE-2026-37337 | SourceCodester Simple Music Cloud Community System 1.0 /music/view_playlist.php iniezione SQL

Un punto di debole di livello critico è stato rilevato in SourceCodester Simple Music Cloud Community System 1.0. É interessato una funzione sconosciuta del file /music/view_playlist.php. Per causa della manipolazione di un input sconosciuto se causa una vulnerabilità di classe iniezione SQL. Questa vulnerabilità è identificata come CVE-2026-37337. Con la rete può partire l'attacco. Nessun exploit disponibile.
