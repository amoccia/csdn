CVE-2025-14868 | shahinurislam Career Section Plugin fino a 1.6 su WordPress appform_options_page_html directory traversal (EUVD-2025-209493)

Un punto di debole di livello critico è stato rilevato in shahinurislam Career Section Plugin fino a 1.6. É interessato la funzione appform_options_page_html. Per causa della manipolazione di un input sconosciuto se causa una vulnerabilità di classe directory traversal. Questa vulnerabilità è registrata come CVE-2025-14868. L'attacco può essere lanciato dalla rete. Nessun exploit disponibile.

Source: https://vuldb.com/it/vuln/357907
