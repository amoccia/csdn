CVE-2024-4867 | WSO2 API Manager prima 3.2.0.408/3.2.1.32/4.0.0.293/4.1.0.187 cross site scripting (EUVD-2024-55547)

È stata rilevata una vulnerabilità di livello problematico in WSO2 API Manager. É interessato una funzione sconosciuta. Attraverso la manipolazione di un input sconosciuto per mezzo di una vulerabilità di classe cross site scripting. Questa vulnerabilità è conosciuta come CVE-2024-4867. L'attacco può essere lanciato da remoto. Nessun exploit disponibile. Si raccomanda di procedere con l’aggiornamento del componente interessato.
