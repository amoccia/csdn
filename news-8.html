CVE-2026-6568 | kodcloud KodExplorer fino a 4.52 Public Share share.class.php initShareOld path directory traversal

In kodcloud KodExplorer fino a 4.52 stata rilevata una vulnerabilità di livello critico. Da questa vulnerabilità è interessato la funzione share.class.php::initShareOld del file /app/controller/share.class.php del componente Public Share Handler. Mediante la manipolazione del parametro path di un input sconosciuto conseguenza di una vulerabilità di classe directory traversal. Questa vulnerabilità è segnalata come CVE-2026-6568. L'attacco si effettua con la rete. Inoltre, è presente un exploit disponibile.

Source: https://vuldb.com/it/vuln/358202
