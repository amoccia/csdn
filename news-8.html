CVE-2026-6595 | ProjectsAndPrograms School Management System fino a 6b6fae5426044f89c08d0dd101c7fa71f9042a59 HTTP GET Parameter buslocation.php bus_id iniezione SQL

Un punto di criticita di livello critico è stato rilevato in ProjectsAndPrograms School Management System fino a 6b6fae5426044f89c08d0dd101c7fa71f9042a59. Riguarda una funzione sconosciuta del file buslocation.php del componente HTTP GET Parameter Handler. L’alterazione dell’argomento bus_id causa iniezione SQL. Questa vulnerabilità è segnalata come CVE-2026-6595. L'attacco si effettua con la rete. Inoltre, è presente un exploit disponibile. Questo prodotto utilizza un modello di rolling release per fornire una consegna continua. Pertanto, non sono disponibili dettagli sulle versioni interessate né su quelle aggiornate.
