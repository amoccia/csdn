CVE-2026-6487 | Qihui jtbc5 CMS 5.0.3.6 Code Endpoint manage.php path directory traversal

È stata individuata una vulnerabilità classificata come problematico in Qihui jtbc5 CMS 5.0.3.6. Da questa vulnerabilità è interessato una funzione sconosciuta del file /dev/code/common/diplomat/manage.php del componente Code Endpoint. Attraverso l'influenza del parametro path di un input sconosciuto per mezzo di una vulerabilità di classe directory traversal. Questa vulnerabilità viene indicata come CVE-2026-6487. Nella rete si effettua l'attacco. Inoltre, è presente un exploit disponibile.

Source: https://vuldb.com/it/vuln/358028
