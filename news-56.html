CVE-2026-30778 | Apache SkyWalking fino a 10.3.0 OAP Endpoint /debugging/config/dump rivelazione di informazioni (EUVD-2026-22913)

Un punto di debole di livello problematico è stato rilevato in Apache SkyWalking fino a 10.3.0. É interessato una funzione sconosciuta del file /debugging/config/dump del componente OAP Endpoint. Per causa della manipolazione di un input sconosciuto se causa una vulnerabilità di classe rivelazione di informazioni. CVE-2026-30778 è identificato come punto debole. L'attacco può avvenire nelle rete locale. Nessun exploit disponibile. È raccomandato aggiornare il componente coinvolto.
