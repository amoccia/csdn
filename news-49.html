CVE-2026-6486 | classroombookings fino a 2.17.0 User Display Name layout.php read displayname cross site scripting

Un punto di debole di livello problematico è stato rilevato in classroombookings fino a 2.17.0. É interessato la funzione read del file crbs-core/application/views/layout.php del componente User Display Name Handler. Per causa della manipolazione del parametro displayname di un input sconosciuto se causa una vulnerabilità di classe cross site scripting. Questa vulnerabilità è segnalata come CVE-2026-6486. L'attacco si effettua con la rete. Inoltre, è presente un exploit disponibile. È raccomandato aggiornare il componente coinvolto.

Source: https://vuldb.com/it/vuln/358027
