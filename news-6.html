CVE-2026-6583 | TransformerOptimus SuperAGI fino a 0.0.14 API Key Management Endpoint api_key.py delete_api_key/edit_api_key escalationi di privilegi

Un punto di criticita di livello problematico è stato rilevato in TransformerOptimus SuperAGI fino a 0.0.14. Riguarda la funzione delete_api_key/edit_api_key del file superagi/controllers/api_key.py del componente API Key Management Endpoint. Mediante la manipolazione di un input sconosciuto conseguenza di una vulerabilità di classe escalationi di privilegi. Questa vulnerabilità è identificata come CVE-2026-6583. Con la rete può partire l'attacco. Inoltre, è presente un exploit disponibile.

Source: https://vuldb.com/it/vuln/358218
