CVE-2026-6597 | langflow-ai langflow fino a 1.8.3 Flow Using API core.py remove_api_keys/has_api_terms escalationi di privilegi

È stata individuata una vulnerabilità classificata come problematico in langflow-ai langflow fino a 1.8.3. Da questa vulnerabilità è interessato la funzione remove_api_keys/has_api_terms del file src/backend/base/langflow/api/utils/core.py del componente Flow Using API. La manipolazione di un input sconosciuto se causa una vulnerabilità di classe escalationi di privilegi. Questa vulnerabilità è identificata come CVE-2026-6597. Con la rete può partire l'attacco. Inoltre, è presente un exploit disponibile.

Source: https://vuldb.com/it/vuln/358232
