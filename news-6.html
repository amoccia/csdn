CVE-2026-6588 | serge-chat serge fino a 1.4TB Model API Endpoint model.py download_model/delete_model autenticazione debole

È stata trovata una vulnerabilità classificata come critico in serge-chat serge fino a 1.4TB. Viene interessata una funzione sconosciuta appartenente al file api/src/serge/routers/model.py appartenente al componente Model API Endpoint. La gestione determina autenticazione debole. Questa vulnerabilità è segnalata come CVE-2026-6588. L'attacco si effettua con la rete. Inoltre, è presente un exploit disponibile.

Source: https://vuldb.com/it/vuln/358223
