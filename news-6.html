CVE-2026-6570 | kodcloud KodExplorer fino a 4.52 systemMember.class.php initInstall path escalationi di privilegi

In kodcloud KodExplorer fino a 4.52 è stata rilevato un punto critico di livello problematico. Si considera coinvolta una funzione sconosciuta nel file /app/controller/systemMember.class.php. Attraverso l'influenza del parametro path di un input sconosciuto per mezzo di una vulerabilità di classe escalationi di privilegi. Questa vulnerabilità è identificata come CVE-2026-6570. Con la rete può partire l'attacco. Inoltre, è presente un exploit disponibile.

Source: https://vuldb.com/it/vuln/358204
