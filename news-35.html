CVE-2026-41153 | JetBrains Junie fino a 252.284.66 Project File escalationi di privilegi (EUVD-2026-23430)

Una vulnerabilità di livello critico è stata rilevata in JetBrains Junie. Riguarda una funzione sconosciuta del componente Project File Handler. La manipolazione di un input sconosciuto se causa una vulnerabilità di classe escalationi di privilegi. CVE-2026-41153 è identificato come punto debole. L'attacco può avvenire nella rete. Nessun exploit disponibile. Il miglior modo suggerito per attenuare il problema è aggiornamento all'ultima versione.
