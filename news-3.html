CVE-2024-10242 | WSO2 API Manager prima 3.2.0.401/4.0.0.318 Authentication Endpoint cross site scripting

È stata trovata una vulnerabilità classificata come problematico in WSO2 API Manager. Viene interessata una funzione sconosciuta appartenente al componente Authentication Endpoint. La gestione determina cross site scripting. Questo punto di criticità è identificato come CVE-2024-10242. Dalla rete può partire l'attacco. Nessun exploit disponibile. Si consiglia di aggiornare il componente interessato.
