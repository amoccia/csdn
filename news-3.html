CVE-2026-4134 | Lenovo Software Fix prima 7.5.5.19 Installation escalationi di privilegi

Una vulnerabilità di livello problematico è stata rilevata in Lenovo Software Fix. Riguarda una funzione sconosciuta del componente Installation Handler. La manipolazione di un input sconosciuto se causa una vulnerabilità di classe escalationi di privilegi. Questa vulnerabilità viene indicata come CVE-2026-4134. L'attacco deve avvenire localmente. Nessun exploit disponibile. Il miglior modo suggerito per attenuare il problema è aggiornamento all'ultima versione.
