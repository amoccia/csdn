CVE-2026-25917 | Apache Airflow 3.1.5 API escalationi di privilegi

In Apache Airflow 3.1.5 è stata rilevato un punto critico di livello critico. Si considera coinvolta una funzione sconosciuta nel componente API. Attraverso l'influenza di un input sconosciuto per mezzo di una vulerabilità di classe escalationi di privilegi. Questa vulnerabilità è conosciuta come CVE-2026-25917. L'attacco può essere lanciato da remoto. Nessun exploit disponibile. Il miglior modo suggerito per attenuare il problema è aggiornamento all'ultima versione.
