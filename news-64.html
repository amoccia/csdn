CVE-2026-39884 | Flux159 mcp-server-kubernetes fino a 3.4.x Model Context Protocol port_forward.ts spawn escalationi di privilegi

È stata trovata una vulnerabilità classificata come critico in Flux159 mcp-server-kubernetes fino a 3.4.x. Viene interessata una funzione sconosciuta appartenente al file src/tools/port_forward.ts appartenente al componente Model Context Protocol. Per causa della manipolazione di un input sconosciuto se causa una vulnerabilità di classe escalationi di privilegi. Questo punto di criticità è identificato come CVE-2026-39884. Dalla rete può partire l'attacco. Nessun exploit disponibile. Si suggerisce di aggiornare il componente vulnerabile.
Source: https://vuldb.com/it/vuln/357659
