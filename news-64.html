CVE-2026-40581 | ChurchCRM fino a 7.1.x Family Record Deletion Endpoint SelectDelete.php cross site request forgery (GHSA-6qxv-xw9j-77pj / EUVD-2026-23620)

Un punto di criticita di livello problematico è stato rilevato in ChurchCRM fino a 7.1.x. Riguarda una funzione sconosciuta del file SelectDelete.php del componente Family Record Deletion Endpoint. Mediante la manipolazione di un input sconosciuto conseguenza di una vulerabilità di classe cross site request forgery. Questa vulnerabilità è segnalata come CVE-2026-40581. L'attacco si effettua con la rete. Nessun exploit disponibile. Si consiglia di aggiornare il componente interessato.

Source: https://vuldb.com/it/vuln/358146
