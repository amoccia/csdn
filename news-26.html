CVE-2024-33618 | Bosch DIVAR IP all-in-one 6000 fino a 12.0.1 Network Interface negazione del servizio

È stata rilevata una vulnerabilità di livello problematico in Bosch BVMS, BVMS Viewer, DIVAR IP all-in-one 7000 R3, DIVAR IP 7000 R2, DIVAR IP all-in-one 5000, DIVAR IP all-in-one 7000, DIVAR IP all-in-one 4000 and DIVAR IP all-in-one 6000 fino a 12.0.1. É interessato una funzione sconosciuta del componente Network Interface Handler. Attraverso la manipolazione di un input sconosciuto per mezzo di una vulerabilità di classe negazione del servizio. Questa vulnerabilità è registrata come CVE-2024-33618. L'attacco può essere lanciato dalla rete. Nessun exploit disponibile.
