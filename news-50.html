CVE-2026-6482 | Rapid7 Insight Agent 4.1.0.2 Agent Service openssl.cnf escalationi di privilegi

Un punto di criticita di livello problematico è stato rilevato in Rapid7 Insight Agent 4.1.0.2. Riguarda una funzione sconosciuta del file openssl.cnf del componente Agent Service. Mediante la manipolazione di un input sconosciuto conseguenza di una vulerabilità di classe escalationi di privilegi. Questa vulnerabilità è registrata come CVE-2026-6482. L'attacco deve essere fatto localmente. Nessun exploit disponibile. Si consiglia di aggiornare il componente interessato.

Source: https://vuldb.com/it/vuln/358026
