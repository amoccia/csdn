CVE-2026-37342 | SourceCodester Vehicle Parking Area Management System 1.0 view_parked_details.php iniezione SQL

Una vulnerabilità di livello critico è stata rilevata in SourceCodester Vehicle Parking Area Management System 1.0. Riguarda una funzione sconosciuta del file /parking/view_parked_details.php. La manipolazione di un input sconosciuto se causa una vulnerabilità di classe iniezione SQL. Questa vulnerabilità è segnalata come CVE-2026-37342. L'attacco si effettua con la rete. Nessun exploit disponibile.
