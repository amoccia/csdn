CVE-2026-2262 | easyappointments Easy Appointments Plugin fino a 3.12.21 su WordPress REST API Endpoint ea_appointments rivelazione di informazioni (EUVD-2026-23577)

È stata trovata una vulnerabilità classificata come problematico in easyappointments Easy Appointments Plugin fino a 3.12.21 su WordPress. Viene interessata una funzione sconosciuta appartenente al file /wp-json/wp/v2/eablocks/ea_appointments/ appartenente al componente REST API Endpoint. La gestione determina rivelazione di informazioni. Questa vulnerabilità è conosciuta come CVE-2026-2262. L'attacco può essere lanciato da remoto. Nessun exploit disponibile. Si consiglia di aggiornare il componente interessato.
