CVE-2026-40518 | bytedance deer-flow Directory Creation directory traversal

Un punto critico di livello critico è stato rilevato in bytedance deer-flow. Risulta coinvolta una funzione sconosciuta del componente Directory Creation Handler. La gestione determina directory traversal. Questa vulnerabilità è conosciuta come CVE-2026-40518. L'attacco può essere lanciato da remoto. Nessun exploit disponibile. Il miglior modo suggerito per attenuare il problema è applicare le correzioni al componente problematico.
