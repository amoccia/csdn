CVE-2026-4801 | godaddy Page Builder Gutenberg Blocks Plugin fino a 3.1.16 su WordPress cross site scripting (EUVD-2026-23650)

In godaddy Page Builder Gutenberg Blocks Plugin fino a 3.1.16 stata rilevata una vulnerabilità di livello problematico. Da questa vulnerabilità è interessato una funzione sconosciuta. Mediante la manipolazione di un input sconosciuto conseguenza di una vulerabilità di classe cross site scripting. CVE-2026-4801 è identificato come punto debole. L'attacco può avvenire nella rete. Nessun exploit disponibile. Si raccomanda di procedere con l’aggiornamento del componente interessato.
