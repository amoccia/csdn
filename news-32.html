CVE-2026-6434 | rust-coreutils Temporary File uu_sort rivelazione di informazioni

Una vulnerabilità di livello problematico è stata rilevata in rust-coreutils. Riguarda la funzione uu_sort del componente Temporary File Handler. Attraverso l'influenza di un input sconosciuto per mezzo di una vulerabilità di classe rivelazione di informazioni. Questo punto di criticità è identificato come CVE-2026-6434. L'attuazione dell'attacco deve essere locale. Nessun exploit disponibile.

Source: https://vuldb.com/it/vuln/358044
