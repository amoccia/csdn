CVE-2026-40259 | SiYuan fino a 3.6.3 removeUnusedAttributeView ID escalationi di privilegi (GHSA-7m5h-w69j-qggg)

È stata trovata una vulnerabilità classificata come critico in SiYuan fino a 3.6.3. Viene interessata una funzione sconosciuta appartenente al file /api/av/removeUnusedAttributeView. Per causa della manipolazione del parametro ID di un input sconosciuto se causa una vulnerabilità di classe escalationi di privilegi. Questo punto di criticità è identificato come CVE-2026-40259. Dalla rete può partire l'attacco. Nessun exploit disponibile. Il miglior modo suggerito per attenuare il problema è aggiornamento all'ultima versione.
