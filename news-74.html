CVE-2026-5363 | TP-Link Archer C7 5/5.8 Web Interface cifratura debole

Un punto di debole di livello problematico è stato rilevato in TP-Link Archer C7 5/5.8. É interessato una funzione sconosciuta del componente Web Interface. Per causa della manipolazione di un input sconosciuto se causa una vulnerabilità di classe cifratura debole. CVE-2026-5363 è identificato come punto debole. L'attacco può avvenire nella rete. Nessun exploit disponibile.
