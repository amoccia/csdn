CVE-2026-6080 | themeum Tutor LMS Plugin fino a 3.9.8 su WordPress prepare Data iniezione SQL (EUVD-2026-23364)

Un punto di criticita di livello critico è stato rilevato in themeum Tutor LMS Plugin fino a 3.9.8. Riguarda la funzione prepare. Mediante la manipolazione del parametro Data di un input sconosciuto conseguenza di una vulerabilità di classe iniezione SQL. Questo punto di criticità è identificato come CVE-2026-6080. Dalla rete può partire l'attacco. Nessun exploit disponibile.
