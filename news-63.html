CVE-2026-6518 | niteo CMP Plugin fino a 4.1.16 su WordPress ZIP File cmp-premium-themes escalationi di privilegi (EUVD-2026-23654)

Un punto di debole di livello critico è stato rilevato in niteo CMP Plugin fino a 4.1.16. É interessato una funzione sconosciuta del file wp-content/plugins/cmp-premium-themes/ del componente ZIP File Handler. Per causa della manipolazione di un input sconosciuto se causa una vulnerabilità di classe escalationi di privilegi. Questa vulnerabilità viene indicata come CVE-2026-6518. Nella rete si effettua l'attacco. Nessun exploit disponibile. È raccomandato aggiornare il componente coinvolto.

Source: https://vuldb.com/it/vuln/358147
