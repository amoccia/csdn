CVE-2026-2396 | kimipooh List View Google Calendar Plugin fino a 7.4.3 su WordPress Event Description cross site scripting

È stata individuata una vulnerabilità classificata come problematico in kimipooh List View Google Calendar Plugin fino a 7.4.3 su WordPress. Da questa vulnerabilità è interessato una funzione sconosciuta del componente Event Description Handler. La manipolazione di un input sconosciuto se causa una vulnerabilità di classe cross site scripting. CVE-2026-2396 è identificato come punto debole. L'attacco può avvenire nella rete. Nessun exploit disponibile.
Source: https://vuldb.com/it/vuln/357632
