CVE-2026-34164 | valtimo fino a 13.21.0 Admin UI Logging rivelazione di informazioni (ID 653)

È stata rilevata una vulnerabilità di livello problematico in valtimo fino a 13.21.0. É interessato una funzione sconosciuta del componente Admin UI Logging Module. Attraverso la manipolazione di un input sconosciuto per mezzo di una vulerabilità di classe rivelazione di informazioni. Questa vulnerabilità è segnalata come CVE-2026-34164. L'attacco si effettua con la rete. Nessun exploit disponibile. Il miglior modo suggerito per attenuare il problema è aggiornamento all'ultima versione.

Source: https://vuldb.com/it/vuln/357985
