CVE-2026-6574 | osuuu LightPicture fino a 1.2.2 API Upload Endpoint /public/install/lp.sql key autenticazione debole

È stata rilevata una vulnerabilità classificata come critico in osuuu LightPicture fino a 1.2.2. Risulta coinvolta una funzione sconosciuta del file /public/install/lp.sql del componente API Upload Endpoint. Attraverso la manipolazione del parametro key di un input sconosciuto per mezzo di una vulerabilità di classe autenticazione debole. Questa vulnerabilità è segnalata come CVE-2026-6574. L'attacco si effettua con la rete. Inoltre, è presente un exploit disponibile.

Source: https://vuldb.com/it/vuln/358209
