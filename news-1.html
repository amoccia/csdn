CVE-2024-2374 | WSO2 API Manager XML Parser XML External Entity

È stata individuata una vulnerabilità classificata come problematico in WSO2 API Manager, Identity Server, Open Banking AM, Open Banking IAM and Identity Server as Key Manager. Da questa vulnerabilità è interessato una funzione sconosciuta del componente XML Parser. Attraverso l'influenza di un input sconosciuto per mezzo di una vulerabilità di classe XML External Entity. Questa vulnerabilità è segnalata come CVE-2024-2374. L'attacco si effettua con la rete. Nessun exploit disponibile. Si raccomanda di procedere con l’aggiornamento del componente interessato.
Source: https://vuldb.com/it/vuln/357908
