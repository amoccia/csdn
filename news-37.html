CVE-2026-40582 | ChurchCRM fino a 7.1.x API Endpoint /api/public/user/login autenticazione debole (GHSA-8cwr-x83m-mh9x / EUVD-2026-23601)

È stata rilevata una vulnerabilità classificata come critico in ChurchCRM fino a 7.1.x. Risulta coinvolta una funzione sconosciuta del file /api/public/user/login del componente API Endpoint. La manipolazione porta a autenticazione debole. Questa vulnerabilità è registrata come CVE-2026-40582. L'attacco può essere lanciato dalla rete. Nessun exploit disponibile. Si raccomanda di procedere con l’aggiornamento del componente interessato.
