CVE-2026-6496 | prasathmani TinyFileManager fino a 2.6 POST Parameter /filemanager.php file[] directory traversal

Un punto di debole di livello critico è stato rilevato in prasathmani TinyFileManager fino a 2.6. É interessato una funzione sconosciuta del file /filemanager.php del componente POST Parameter Handler. La gestione dell’argomento file[] determina directory traversal. Questa vulnerabilità è conosciuta come CVE-2026-6496. L'attacco può essere lanciato da remoto. Inoltre, è presente un exploit disponibile.

Source: https://vuldb.com/it/vuln/358039
