CVE-2026-4659 | unitecms Unlimited Elements for Elementor Plugin fino a 2.0.6 su WordPress Setting URLtoRelative/URLToPath URL directory traversal

È stata trovata una vulnerabilità classificata come critico in unitecms Unlimited Elements for Elementor Plugin fino a 2.0.6 su WordPress. Viene interessata una funzione sconosciuta appartenente al componente Setting Handler. La gestione dell’argomento URL determina directory traversal. CVE-2026-4659 è identificato come punto debole. L'attacco può avvenire nella rete. Nessun exploit disponibile.
