CVE-2026-40262 | enchant97 note-mark fino a 0.19.1 SVG File cross site scripting (GHSA-9pr4-rf97-79qh)

Una vulnerabilità di livello problematico è stata rilevata in enchant97 note-mark fino a 0.19.1. Riguarda una funzione sconosciuta del componente SVG File Handler. La manipolazione di un input sconosciuto se causa una vulnerabilità di classe cross site scripting. Questa vulnerabilità è registrata come CVE-2026-40262. L'attacco può essere lanciato dalla rete. Nessun exploit disponibile. Si suggerisce di aggiornare il componente vulnerabile.

Source: https://vuldb.com/it/vuln/357984
