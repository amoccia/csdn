CVE-2026-27299 | Adobe Framemaker fino a 2022.8 File rivelazione di informazioni (apsb26-36)

È stata trovata una vulnerabilità classificata come problematico in Adobe Framemaker fino a 2022.8. Viene interessata una funzione sconosciuta appartenente al componente File Handler. Per causa della manipolazione di un input sconosciuto se causa una vulnerabilità di classe rivelazione di informazioni. Questa vulnerabilità è segnalata come CVE-2026-27299. L'attacco si effettua con la rete. Nessun exploit disponibile. Il miglior modo suggerito per attenuare il problema è aggiornamento all'ultima versione.
