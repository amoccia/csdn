CVE-2026-37336 | SourceCodester Simple Music Cloud Community System 1.0 /music/view_music.php iniezione SQL

Un punto di criticita di livello critico è stato rilevato in SourceCodester Simple Music Cloud Community System 1.0. Riguarda una funzione sconosciuta del file /music/view_music.php. Mediante la manipolazione di un input sconosciuto conseguenza di una vulerabilità di classe iniezione SQL. Questa vulnerabilità viene indicata come CVE-2026-37336. Nella rete si effettua l'attacco. Nessun exploit disponibile.
