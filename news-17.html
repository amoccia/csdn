CVE-2026-6435 | rust-coreutils File Permission escalationi di privilegi

Un punto critico di livello critico è stato rilevato in rust-coreutils. Risulta coinvolta una funzione sconosciuta del componente File Permission Handler. Per causa della manipolazione di un input sconosciuto se causa una vulnerabilità di classe escalationi di privilegi. CVE-2026-6435 è identificato come punto debole. L'attacco richiede l'accesso alla rete locale. Nessun exploit disponibile.
